PROJECT_01
🐙 GitHub

❤️ CVD Risk Predictor

Objective

Build a machine learning system to predict Cardiovascular Disease (CVD) risk levels — Low, Medium, or High — based on patient health parameters such as age, BMI, cholesterol, and blood sugar. The goal was to aid early risk assessment and preventive healthcare decisions.

What I Did
  • ▸Processed a large Kaggle dataset of 308,854 patient records with 15+ health features
  • ▸Cleaned data: handled missing values (median/mode imputation), parsed blood pressure, IQR outlier winsorization
  • ▸Applied one-hot encoding for categorical features and label encoding for the 3-class target
  • ▸Trained and compared 4 models: Logistic Regression, Decision Tree, Random Forest (with/without SMOTE), and XGBoost
  • ▸Deployed the full system as an interactive Streamlit web app with real-time prediction and model comparison UI
Results

XGBoost — Best Model 67.65% Accuracy 308K+ Records Processed 4 Models Compared Streamlit App Deployed

XGBoost achieved the highest accuracy of 67.65% on this challenging multi-class imbalanced dataset. Key challenge was detecting the minority High-risk class — addressed using SMOTE oversampling. Final app allows users to input patient vitals and instantly get risk level predictions with model comparison.

PROJECT_02
🐙 GitHub

📈 Trader Behavior vs Market Sentiment

Objective

Analyze how market sentiment (Fear vs Greed) correlates with trader behavior and performance on the Hyperliquid platform. The goal was to uncover patterns that link market psychology to measurable trading outcomes and inform smarter risk management strategies.

What I Did
  • ▸Combined two datasets: Fear & Greed Index (daily sentiment score 0–100) and Hyperliquid historical trading data
  • ▸Converted timestamps to daily format and merged on date column to create unified analysis dataset
  • ▸Created sentiment buckets: Fear (0–40), Neutral (40–60), Greed (60–100) and mapped trader behavior per regime
  • ▸Analysed profitability differences, risk-taking patterns, and activity levels across all 3 sentiment regimes
  • ▸Segmented traders by behavior consistency and translated findings into actionable sentiment-aware trading rules
Results

3 Sentiment Regimes Analyzed Fear vs Greed Patterns Found Trader Segments Identified Actionable Strategy Rules

Analysis revealed clear behavioral shifts: traders take on higher risk during Greed phases and reduce activity in Fear periods. Segment analysis identified consistent vs reactive trader types. Results translated into sentiment-aware risk adjustment rules that could improve trading strategy design.
