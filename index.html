Featured Work

PROJECT_01
🐙 GitHub
❤️ CVD Risk Predictor
Objective

Build an ML system to predict Cardiovascular Disease risk — Low, Medium, or High — based on patient health parameters. Aimed to aid early risk assessment and preventive healthcare.

What I Did
  • ▸Processed 308,854 patient records with 15+ health features from Kaggle
  • ▸Cleaned data: median/mode imputation, IQR outlier winsorization, feature encoding
  • ▸Trained and compared 4 models: Logistic Regression, Decision Tree, Random Forest, XGBoost
  • ▸Addressed class imbalance using SMOTE oversampling on the minority High-risk class
  • ▸Deployed as an interactive Streamlit app with real-time prediction & model comparison
Results
XGBoost — Best Model67.65% Accuracy308K+ Records4 Models ComparedStreamlit Deployed

PROJECT_02
🐙 GitHub
📈 Trader Behavior vs Market Sentiment
Objective

Analyze how market sentiment (Fear vs Greed) correlates with trader behavior on Hyperliquid. Uncover patterns linking market psychology to measurable trading outcomes.

What I Did
  • ▸Combined Fear & Greed Index with Hyperliquid historical trading data
  • ▸Created sentiment buckets: Fear (0–40), Neutral (40–60), Greed (60–100)
  • ▸Analyzed profitability differences and risk-taking patterns across all 3 regimes
  • ▸Segmented traders by behavior consistency across sentiment regimes
  • ▸Translated findings into actionable sentiment-aware trading strategy rules

PROJECT_03
🐙 GitHub 🌐 Live
🌍 Vibe & Go — Mood-Based Place Discovery
Objective

A full-stack web app that recommends nearby places based on the user's current mood. Built with Next.js, MongoDB, and AI-powered suggestions using Groq LLaMA — deployed on Vercel with full authentication and OTP email verification.

What I Built
  • ▸Full auth system with JWT tokens, bcrypt password hashing and email OTP verification via Gmail SMTP
  • ▸AI mood detection using Groq LLaMA 3.3 — user describes their vibe, AI picks the best mood category
  • ▸Real-time place search using OpenStreetMap Overpass API filtered by 9 mood categories
  • ▸Save favourite places to MongoDB, mood journal with localStorage, weather-aware suggestions
  • ▸CI/CD pipeline with GitHub Actions — auto deploys to Vercel on every push to main
